Список примеров. Краткие описания

  1. Программа поиска значения максимального элемента целочисленного массива из n элементов (n<=100)
  2. Программа подсчета отрицательных чисел в массиве из n (n<=100) вещественных чисел
  3. Программа сортировки массива из n вещественных чисел (n<=100) по возрастанию методом прямого обмена (пузырьковым методом)
  4. Программа удаления из массива, содержащего n целых чисел, последнего отрицательного элемента
  5. Программа формировния нового массива из отрицательных чисел целочисленного массива размера n (n<=100)
  6. Программа вычисления суммы чисел, расположенных выше главной диагонали матрицы целых чисел из n строк и n столбцов (n<=100)
