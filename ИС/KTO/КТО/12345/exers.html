  2. В трехзначном числе зачеркнули первую цифру, полученное число умножили на 7 и получили исходное число. Найдите его.

  3. По данному натуральному d найдите наименьшее решение в натуральных числах уравнения Пелля x^2-dy^2=1.

  4. Для данного натурального числа n выведите на экран все его натуральные делители.

  5. По данному n найдите суммы 1+2+...+n, 12+22+...+n2 . Сравните полученные результаты с формулами, известными из курса математического анализа.
  6. Напечатайте все точные квадраты натуральных чисел, не превосходящие данного числа n.
  7. По данному числу n вычислите число n! .
  8. По данным числам n и k вычислите число Ckn (число сочетаний из n элементов по k).
  9. По данному числу n вычислите сумму 4(1-1/3+1/5-1/7+...+(-1)n1/(2n+1)) . К чему будет стремиться эта сумма при росте n?
  10. Найдите наибольшее число, которое можно сохранить в переменных типа short, unsigned short, int, unsigned int, long, unsigned long (подсказка: при прибавлении к этому числу единицы получится меньшее число).
  11. По данному натуральному числу n определите количество цифр в его десятичной записи. Определите сумму цифр числа n. Найдите наибольшую и наименьшую цифры.
  12. Напечатайте представление данного натурального числа n в двоичном виде (можно в обратном порядке).
